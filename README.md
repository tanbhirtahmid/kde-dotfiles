# Fedora KDE Plasma Dotfiles

This repository contains my personal KDE Plasma configuration and themes. The goal is to avoid manual setup after a fresh Fedora installation.

## 📁 What's Included?
* **Themes:** Global themes, Aurorae window decorations, and Icon sets (`.local/share/`).
* **UI Configs:** Panel layouts, Widget settings, and Wallpaper paths (`plasma-org.kde.plasma.desktop-appletsrc`).
* **Shortcuts:** Custom Global Shortcuts (`kglobalshortcutsrc`).
* **Terminal:** Konsole profiles and color schemes.
* **Window Manager:** KWin rules and effects (`kwinrc`).

## 🛡️ Privacy & Sanitation
This repository has been sanitized to remove sensitive information. The following have been excluded:
- Browser profiles (Brave/Chrome/Firefox)
- Email account details (Akonadi/KMail)
- Password databases (KWallet)
- Personal data (Contacts/Trash/History)
- Hardware-specific configs (PulseAudio/Display Layouts)

## 🚀 How to Restore

    Copy and paste everything from here to the home folder (except for the `README.md` file).
